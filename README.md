# 📌 Project Overview

This project presents an **interactive Business Intelligence Dashboard** developed in **Microsoft Power BI** to analyze sales performance for **Big Bazar**.

The dashboard enables management to monitor business performance through dynamic visualizations, KPI tracking, customer analysis, regional sales distribution, and product profitability.

It transforms raw sales data into meaningful business insights that support data-driven decision making.
